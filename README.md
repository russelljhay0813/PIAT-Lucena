# PIAT Academic Management System

## Overview

This repository contains the PIAT Academic Management System, a multi-platform academic operations project with:

- a web frontend powered by Vite and React
- a Node/Express backend with a SQLite database
- a mobile Expo client for attendance and mobile workflows

## Repository structure

- `backend/` — REST API, SQLite database, seeding scripts, and backend documentation
- `mobile_build/` — Expo mobile client and mobile README
- `src/` — web application source files and route definitions
- `package.json` — root project dependencies and any root-level scripts

## Quick start

1. Install root dependencies:
   ```bash
   npm install
   ```
2. Start the backend service:
   ```bash
   cd backend
   npm install
   npm start
   ```
3. Start the web frontend:
   ```bash
   cd ..
   npm run dev
   ```
4. Start the mobile app (optional):
   ```bash
   cd mobile_build
   npm install
   npm start
   ```

## Backend and data seeding

- The backend uses SQLite and stores data in `backend/bwest.db`.
- Generated student records can be seeded using:
  ```bash
  cd backend
  node seed-students.mjs
  ```
- The seed script creates 900 student records and associated student user accounts.

## Notes

- The backend API runs on `http://localhost:4000` by default.
- The web frontend should proxy API requests to the backend.
- The mobile app requires a running backend and properly configured API base URL.
- The `.expo` folder is generated by Expo and should not be committed.
